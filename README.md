# JavaScript Todo App
A simple Todo List application built with HTML, CSS and JavaScript.

## ✨ Features
- Add new tasks
- Delete tasks
- Mark tasks as complete
- Tasks saved in Local Storage
- Data persists after page refresh

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
- Local Storage

## How It Works
- Users can add tasks using the input field
- Tasks are displayed dynamically using DOM manipulation
- Clicking a task marks it as completed
- Delete button removes tasks
- localStorage stores tasks in the browser
